SquirrelZip
===========

A archive reader/writer/creator in Squirrel

Tested on:
----------

- Half-Life 2: Episode Two
- Portal 2
- Source SDK Base 2013 Singleplayer

Supported archive formats:
--------------------------

+--------+----------------+--------+------+-------+
| Format | Description    | Create | Read | Write |
+========+================+========+======+=======+
| TAR    | Tarball        | ✅      | ✅    | ✅     |
+--------+----------------+--------+------+-------+
| VPK    | Valve PaK      | ✅      | ✅    | ✅     |
+--------+----------------+--------+------+-------+
| ZIP    | ZIP File       | ❌      | ✅    | ✅     |
+--------+----------------+--------+------+-------+
| GZ     | Gzip           | ❌      | ✅    | ❌     |
+--------+----------------+--------+------+-------+
