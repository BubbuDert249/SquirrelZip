# SquirrelZip
A archive reader/writer/creator in Squirrel <br>
Tested on: <Br>
Half-Life 2: Episode Two <br>
Portal 2 <Br>
Source SDK Base 2013 Singleplayer <br>
Supported archive formats: <br>
| Format   | Description | Create | Read | Write |
|----------|----------|----------|------|-------|
| TAR | Tarball | ✅ | ✅ | ✅ |
| VPK | Valve PaK | ✅ | ✅ | ✅ |
| ZIP | ZIP File | ❌ | ✅ | ✅ |
| GZ | Gzip | ❌ | ✅ | ❌ |
